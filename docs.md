### 🚀 快速開始 / Quick Start

#### 安裝 / Installation
```bash
# Windows
.\install.bat

# Manual installation
pip install -r requirements.txt
```

#### 執行 / Run
```bash
# Windows
.\run.bat

# Manual run
python main.py
```
